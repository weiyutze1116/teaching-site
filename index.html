C++ 程式設計：全方位教學指南

從零基礎語法到高階物件導向，建立最紮實的資工底層實力。

Ch1 基礎 Ch2 流程 Ch3 函式 Ch4 陣列 Ch5 指標 Ch6 物件 Ch7 STL Ch8 進階

第 1 章：基礎語法與變數

一切的開始。C++ 是一個編譯型語言，嚴謹的資料型態是其特徵。

#include <iostream>
using namespace std;

int main() {
    int score = 100;       // 整數
    float temp = 36.5f;    // 單精度浮點數
    double pi = 3.14159;   // 雙精度浮點數
    char initial = 'C';    // 單一字元
    bool isPass = true;    // 布林值
    
    cout << "分數為：" << score << endl;
    return 0;
}

第 2 章：流程控制邏輯

程式的靈魂在於「判斷」與「重複」。

條件判斷 (if / else / switch)

if (age >= 18) {
    cout << "成年人";
} else if (age > 12) {
    cout << "青少年";
} else {
    cout << "兒童";
}

迴圈控制 (for / while / do-while)

for (int i = 1; i <= 5; i++) {
    cout << "第 " << i << " 次執行" << endl;
}

第 3 章：函式與作用域

將重複的程式碼打包，實現模組化。包含重要的「傳值」與「傳址」觀念。

💡 重點： C++ 預設是 Pass by Value，若要修改原始變數需使用 Reference (&)。 
// 函式宣告與定義
int add(int a, int b) {
    return a + b;
}

// 傳址呼叫 (Pass by Reference)
void updateValue(int &n) {
    n = 100; 
}

第 4 章：陣列、字串與結構體

處理大量資料的基礎工具。

一維與多維陣列

int nums[5] = {1, 2, 3, 4, 5};
int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};

結構體 (Struct)

struct Student {
    string name;
    int grade;
};

Student s1 = {"小明", 90};

第 5 章：指標與動態記憶體

C++ 與其他高階語言最大的差別，在於直接操控記憶體的能力。

int x = 10;
int* p = &x; // p 儲存 x 的位址

// 動態配置 (Heap)
int* arr = new int[10]; 
delete[] arr; // 務必釋放，否則會產生 Memory Leak

第 6 章：物件導向 (OOP)

封裝、繼承與多型：現代軟體開發的基石。

class Shape {
public:
    virtual void draw() { cout << "繪製圖形"; } // 多型
};

class Circle : public Shape {
public:
    void draw() override { cout << "繪製圓形"; }
};

第 7 章：STL 標準模板庫

不要重新造輪子，學會使用 C++ 強大的內建工具。

#include <vector>
#include <map>
#include <algorithm>

vector<int> v = {5, 2, 8};
sort(v.begin(), v.end()); // 排序
v.push_back(10);          // 新增元素

第 8 章：進階主題 (檔案與異常)

如何處理錯誤以及讀寫檔案。

#include <fstream>

// 寫入檔案
ofstream outfile("test.txt");
outfile << "寫入內容";
outfile.close();

// 異常處理
try {
    if (denominator == 0) throw "除數不可為零";
} catch (const char* msg) {
    cerr << msg << endl;
}
