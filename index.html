📘 C++ 程式設計完全指南

這是一份系統化的 C++ 學習路徑，專為想要掌握程式邏輯的初學者設計。

快速跳轉： 基礎語法 流程控制 函式與陣列 指標與記憶體 物件導向

第 0 章：基本程式架構

每一段 C++ 程式都從 main 函式開始執行。

#include <iostream> // 引入標準輸入輸出庫

int main() {
    // std::cout 代表輸出到螢幕
    std::cout << "Hello C++!" << std::endl; 
    return 0; // 程式正常結束
}

第 1 章：變數與資料型態

C++ 是一種強型別語言，每個變數都必須宣告型別。

  • int: 整數 (例如 10, -5)
  • double: 浮點數 (例如 3.14)
  • char: 單一字元 (例如 'A')
  • bool: 布林值 (true / false)
int age = 20;
double price = 99.9;
bool isReady = true;

第 2 章：條件判斷與迴圈

控制程式的執行路徑是邏輯開發的核心。

條件判斷 (If-Else)

if (age >= 18) {
    std::cout << "成年人";
} else {
    std::cout << "未成年";
}

重複執行 (Loops)

// 印出 0 到 4
for (int i = 0; i < 5; i++) {
    std::cout << i << " ";
}

第 3 章：指標 (Pointers) - C++ 的靈魂

指標是儲存記憶體位址的變數，這是 C++ 最強大也最危險的地方。

int val = 10;
int* ptr = &val; // ptr 儲存了 val 的位址

std::cout << *ptr; // 取值符號 (*)，輸出 10

第 4 章：物件導向 (OOP)

透過類別 (Class) 將資料與行為封裝在一起。

class Dog {
public:
    void bark() {
        std::cout << "汪汪！";
    }
};

int main() {
    Dog myDog;
    myDog.bark(); // 呼叫物件的方法
}
