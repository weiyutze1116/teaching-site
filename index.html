第 1 章：基礎語法與資料型態

了解如何與電腦溝通，從變數宣告開始。

核心重點： 變數宣告、輸入輸出 (cin/cout)、常數 (const)。 
#include <iostream>
using namespace std;

int main() {
    int age = 20;          // 整數
    double pi = 3.14159;   // 浮點數
    char grade = 'A';      // 字元
    string name = "User";  // 字串 (需包含 <string>)
    
    cout << "請輸入一個數字：";
    cin >> age;
    return 0;
}

第 2 章：流程控制

讓程式具備判斷能力與重複執行任務的能力。

條件判斷 (if / switch)

if (score >= 60) {
    cout << "及格";
} else {
    cout << "不及格";
}

迴圈控制 (for / while / do-while)

// for 迴圈常用於已知次數
for (int i = 0; i < 10; i++) {
    cout << i << endl;
}

第 5 章：指標與記憶體管理 (Pointers)

資工核心： 這是 C++ 最強大也最難的部分，允許直接存取記憶體位址。 
int x = 10;
int* ptr = &x; // ptr 指向 x 的記憶體位址

cout << "位址：" << ptr << endl;
cout << "內容：" << *ptr << endl; // 解引用 (Dereference)

// 動態記憶體配置
int* dynamicArr = new int[5]; // 在 Heap 區塊要空間
delete[] dynamicArr;          // 務必手動釋放防止 Memory Leak

第 6 章：物件導向程式設計 (OOP)

將真實世界的概念抽象化為類別 (Class) 與物件 (Object)。

  • 封裝 (Encapsulation)： 使用 private/public 保護資料。
  • 繼承 (Inheritance)： 子類別繼承父類別屬性。
  • 多型 (Polymorphism)： 同一個介面，多種實現。
class Animal {
public:
    virtual void makeSound() { cout << "某種聲音"; }
};

class Cat : public Animal {
public:
    void makeSound() override { cout << "喵喵"; }
};

第 7 章：STL 標準模板庫

資工系實作中最常用的工具箱。

#include <vector>
#include <algorithm>

vector<int> v = {3, 1, 4};
v.push_back(2);    // 動態新增元素
sort(v.begin(), v.end()); // 內建排序演算法
