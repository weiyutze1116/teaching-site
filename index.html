第 1 章：哈囉 C++！我的第一隻程式

寫程式就像是在下指令。這一章我們要學會如何讓電腦說話，以及認識電腦如何「儲存」資料。

1.1 程式的基本架構

#include <iostream> // 引入「輸入輸出」功能
                using namespace std; // 小祕技：讓我們不用一直打 std::
                
                int main() {
                    // cout 是輸出的指令，endl 是換行
                    cout << "你好，C++！" << endl; 
                    return 0;
                }

1.2 置物櫃詳解：認識常用的資料型態

變數就像是貼了標籤的置物櫃，不同櫃子要裝不同的東西：

📦 櫃子 A：整數家族 (int / long long)

當你只需要存數字，而且沒有小數點時使用。

  • int： 夠用就好（約可存正負 21 億）。
  • long long： 超大數字專用（例如：全地球的人口）。
int age = 17;
        long long bigNumber = 9000000000000000000LL;

🍕 櫃子 B：小數家族 (double / float)

需要計算精確數值（如體重、GPA、圓周率）時使用。

  • double： 最常用，精確度高。
  • float： 較省空間，但精確度較低（數字後面要加 f）。
double height = 175.5;
        float weight = 65.2f;

🔠 櫃子 C：文字家族 (char / string)

注意！單引號 ' ' 跟雙引號 " " 是完全不同的意義喔！

  • char： 單一字元（只能放一個字母）。
  • string： 一串文字。
char grade = 'A';
        string school = "Yilan Senior High School";

1.3 與電腦對話 (cin)

我們可以使用 cin 來讓使用者從鍵盤輸入資料到置物櫃裡：

string userName;
        cout << "請問你叫什麼名字？ ";
        cin >> userName;
        cout << "你好，" << userName << "！歡迎來到 C++ 的世界。" << endl;
💡 小撇步：endl vs \n
endl 除了換行，還會強迫電腦「清空緩衝區」確保文字立刻印出來；而 \n 單純就是換行，速度稍微快一點點。

1.4 數學運算：不只是加減乘除

除了常見的運算，高中生最常卡關的是「餘數」運算：

int a = 10;
        int b = 3;
        
        cout << a + b; // 加法：13
        cout << a - b; // 減法：7
        cout << a * b; // 乘法：30
        cout << a / b; // 除法：3 (因為是 int，小數會不見)
        cout << a % b; // 取餘數：1 (10 除以 3 餘 1)

第 2 章：流程控制邏輯

程式的靈魂在於「判斷」與「重複」。

條件判斷 (if / else / switch)

if (age >= 18) {
    cout << "成年人";
} else if (age > 12) {
    cout << "青少年";
} else {
    cout << "兒童";
}

迴圈控制 (for / while / do-while)

for (int i = 1; i <= 5; i++) {
    cout << "第 " << i << " 次執行" << endl;
}

第 3 章：函式與作用域

將重複的程式碼打包，實現模組化。包含重要的「傳值」與「傳址」觀念。

💡 重點： C++ 預設是 Pass by Value，若要修改原始變數需使用 Reference (&)。 
// 函式宣告與定義
int add(int a, int b) {
    return a + b;
}

// 傳址呼叫 (Pass by Reference)
void updateValue(int &n) {
    n = 100; 
}

第 4 章：陣列、字串與結構體

處理大量資料的基礎工具。

一維與多維陣列

int nums[5] = {1, 2, 3, 4, 5};
int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};

結構體 (Struct)

struct Student {
    string name;
    int grade;
};

Student s1 = {"小明", 90};

第 5 章：指標與動態記憶體

C++ 與其他高階語言最大的差別，在於直接操控記憶體的能力。

int x = 10;
int* p = &x; // p 儲存 x 的位址

// 動態配置 (Heap)
int* arr = new int[10]; 
delete[] arr; // 務必釋放，否則會產生 Memory Leak

第 6 章：物件導向 (OOP)

封裝、繼承與多型：現代軟體開發的基石。

class Shape {
public:
    virtual void draw() { cout << "繪製圖形"; } // 多型
};

class Circle : public Shape {
public:
    void draw() override { cout << "繪製圓形"; }
};

第 7 章：STL 標準模板庫

不要重新造輪子，學會使用 C++ 強大的內建工具。

#include <vector>
#include <map>
#include <algorithm>

vector<int> v = {5, 2, 8};
sort(v.begin(), v.end()); // 排序
v.push_back(10);          // 新增元素

如何處理錯誤以及讀寫檔案。

// 寫入檔案
ofstream outfile("test.txt");
outfile << "寫入內容";

// 異常處理
try {
    if (denominator == 0) throw "除數不可為零";
